
打开浏览器访问: http://localhost:3000

### 3. 创建账户

- 填写邮箱和密码
- 创建账户

## 🔧 常用命令

### 查看日志
```bash
docker-compose logs -f
```

### 停止服务
```bash
docker-compose down
```

### 重启服务
```bash
docker-compose restart
```

### 连接数据库
```bash
docker exec -it blueprint_postgres psql -U postgres -d blueprint_saas
```
