## 💡 重要提示

### 1. 环境变量

部署时需要配置以下环境变量：

**后端**：
```bash
DATABASE_URL=${{Postgres.DATABASE_URL}}
JWT_SECRET=<随机字符串>
NODE_ENV=production
PORT=5000
```

**前端**：
```bash
VITE_API_URL=https://后端地址.railway.app/api
```

### 2. JWT Secret 生成

运行以下命令生成：
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

### 3. CORS 配置

后端需要配置前端域名：
```bash
CORS_ORIGIN=https://前端地址.railway.app
```
