# 🚀 快速开始

> 5 分钟快速体验蓝图 AI SaaS 版本

## ✅ 前置要求

- Windows 10/11
- 已安装 Docker Desktop（[安装指南](./02-Docker安装.md)）

## 🎯 三步启动

### 第一步：配置环境变量

项目已经自动生成了安全的配置，无需手动修改。

### 第二步：启动服务

```bash
# 在项目根目录运行
docker-compose up -d
```

等待 3-5 分钟，Docker 会自动：
- 下载所需镜像
- 构建前后端容器
- 启动数据库
- 初始化数据表

### 第三步：访问应用

打开浏览器访问：**http://localhost:3000**

## 🎉 首次使用

### 注册账户

1. 页面会自动弹出登录窗口
2. 点击"还没有账户？立即注册"
3. 填写信息：
   - 姓名：你的名字
   - 邮箱：your@email.com
   - 密码：至少6位
4. 点击"注册"按钮

### 或使用测试账户

我们已经创建了一个测试账户：
- **邮箱**: test@example.com
- **密码**: 123456

## ✨ 开始使用

登录成功后，你可以：

1. **创建项目** - 点击"项目管理中心"
2. **添加模块** - 定义功能模块
3. **设计实体** - 创建数据模型
4. **使用 AI** - AI 辅助分析

## 📊 验证部署

运行以下命令检查服务状态：

```bash
# 查看容器状态
docker-compose ps

# 应该看到 3 个容器都在运行：
# - blueprint_postgres (数据库)
# - blueprint_backend  (后端 API)
# - blueprint_frontend (前端界面)
```

## 🔍 常用命令

```bash
# 查看日志
docker-compose logs -f

# 重启服务
docker-compose restart

# 停止服务
docker-compose down

# 备份数据库
docker exec blueprint_postgres pg_dump -U postgres blueprint_saas > backup.sql
```
